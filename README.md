# Babushka
Babushka is a Rust library for 2D geometry algorithms, with a focus on efficient bin packing and nesting.

## Features
- No Fit Polygon implementation inspired by SVGNest

## Installation

Add this to your `Cargo.toml`:

[dependencies]
babushka = "0.1.5"
